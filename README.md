# Manual Reader

Welcome to the Manual Reader project! This application allows users to browse and read manuals for various products or services. It provides a user-friendly interface for accessing manuals, searching for specific topics, and managing favorites.

## Features

- Browse manuals: Users can browse through a list of available manuals.
- Search: Users can search for specific topics or keywords within manuals.
- Favorites: Users can mark manuals as favorites for easy access.
- Firebase integration: Backend services are powered by Firebase, including Firestore for data storage and Firebase Authentication for user management.
- Multi-platform support: The frontend is developed using both Kotlin and Java languages, providing flexibility and compatibility with different platforms.

## Technologies Used

- Frontend: Kotlin, Java
- Backend: Firebase (Firestore for database, Firebase Authentication for user authentication)
